# 大数据题目

1. 哈希函数可以把数据按照种类均匀分流
2. 布隆过滤器（有失误）用于集合的建立与查询，并可以节省大量空间
3. 一致性哈希解决数据服务器的负载管理问题
4. 利用并查集结构做岛问题的并行计算
5. 位图（1bit表示一个数出现或者没出现）解决某一范围上数字的出现情况，并可以节省大量空间
6. 利用分段统计思想、并进一步节省大量空间
7. 利用堆、外排序来做多个处理单元的结果合并
